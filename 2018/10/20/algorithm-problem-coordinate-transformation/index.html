坐标跳跃

问题描述

在一个平面坐标系上，小明的初始位置坐标为 (p,q)(p,q)(p,q)，我们可以通过一次操作将他的位置变为 (p+q,q)(p+q,q)(p+q,q) 或 (p,p+q)(p, p+q)(p,p+q)。

给定一正整数 nnn，小明的初始位置为 (1,1)(1, 1)(1,1)，问最少需要多少次操作，可变为一个新的坐标 (x,y)(x, y)(x,y)，且其中的 xxx 或 yyy 至少有一个数字为 nnn 。

输入

第一行一个正整数 nnn

输出

一个整数表示答案

100%100\%100%的数据，1⩽n⩽1061\leqslant n\leqslant 10^61⩽n⩽106

数论相关

分析

观察变换，不难发现，若 gcd⁡(p,q)=1\gcd(p,q)=1gcd(p,q)=1 则有 gcd⁡(p+q,q)=1\gcd(p+q,q)=1gcd(p+q,q)=1.
则最终含有 nnn 的坐标 (x,n)(x,n)(x,n) 中，也必有 gcd⁡(x,n)=1\gcd(x,n)=1gcd(x,n)=1.
而从 (1,1)(1,1)(1,1) 变换到 (x,n)(x,n)(x,n) 的次数，实际上就是计算 gcd⁡(x,n)\gcd(x,n)gcd(x,n) 时辗转相减的次数.
又由于对称性，可在 [2,n2][2,\frac{n}{2}][2,2n​] 中遍历所有 xxx，满足 gcd⁡(x,n)=1\gcd(x,n)=1gcd(x,n)=1，取辗转相减次数最小的即可.
