Algorithm - Basic - Merge Sort

发表于 2018-09-14 | 更新于 2018-10-03 | 分类于 Algorithm ， Basic

归并排序是一种分治排序算法，还可用于求逆序对.

思想

已知一个数组aaa，对于区间[l,r][l, r][l,r]，我们采取操作 MERGE(a,l,r)\mathrm{MERGE}(a, l, r)MERGE(a,l,r)

  1. m←(l+r)/2m\leftarrow (l + r) / 2m←(l+r)/2.
  2. MERGE(a,l,m)\mathrm{MERGE}(a, l, m)MERGE(a,l,m).
  3. MERGE(a,m+1,r)\mathrm{MERGE}(a, m + 1, r)MERGE(a,m+1,r).
  4. 合并，拷回原数组 aaa.

可视化算法请看 VisuAlgo Sorting.

复杂度分析

根据思想，我们可有递推式 T(n)=2T(n2)+O(n)T(n) = 2T(\frac{n}{2}) + \mathcal{O}(n)T(n)=2T(2n​)+O(n).
根据主定理，其时间复杂度为 O(nlog⁡n)\mathcal{O}(n\log n)O(nlogn).
由于合并需要外部空间，其空间复杂度为 O(n)\mathcal{O}(n)O(n).

实现思路

采用递归的方式，参数为数组 aaa，区间左端 lll，区间右端 rrr.
显然在 l⩾rl \geqslant rl⩾r 时不需要进行排序，将其作为递归边界.
记 mmm 为区间中点 (l+r)/2(l + r) / 2(l+r)/2，并对左半部分排序，右半部分排序.
实现的重点实际上是合并左半部分和右半部分，并拷回原数组 aaa.

假定数组 aaa 的 [l,m][l, m][l,m] 和 [m+1,l][m + 1, l][m+1,l] 区间已完成排序，如图所示，它们的合并更像是两个栈出栈的过程.
分别用 lilili 和 ririri 表示左半部分栈顶元素下标和右半部分栈顶元素下标，初始时 li←lli \leftarrow lli←l 且 ri←m+1ri \leftarrow m + 1ri←m+1.
接下来分为两种情况出栈

  1. 选择 a[li]a[li]a[li] 和 a[ri]a[ri]a[ri] 中较小的出栈至外部空间 bufbufbuf，并相应使下标自增
  2. 一个栈已空，则只对另一个栈进行出栈即可

反复进行上述两个操作之一，直到两个栈为空即 li>mli > mli>m 且 ri>rri > rri>r （或使用计数器 cnt\mathrm{cnt}cnt 统计已出栈的个数，并将 cnt==(r−l+1)\mathrm{cnt == (r - l + 1)}cnt==(r−l+1) 作为终止条件）时.
再将 bufbufbuf 的 [l,r][l, r][l,r] 区间拷回原数组 aaa 中.

逆序对

逆序对的计算发生在合并的过程中，由于两部分已经有序，在左半部分和右半部分分别出栈时，可以保证当前出栈的元素一定是小于另一个栈剩下所有元素的.
因此当右半部分出栈时，出栈的元素小于剩余的左半部分的所有元素，而此时左半部分剩余元素个数为 m−li+1m - li + 1m−li+1
