Algorithm - Problem - 二叉树之中序后序求先序

发表于 2018-09-20 | 更新于 2018-09-23 | 分类于 Algorithm ， Problem

给出一棵二叉树的中序与后序遍历，求出它的先序遍历，用不同的大写字母表示二叉树的结点.

分析

记 XXX 为一个结点，LLL 为其左孩，RRR 为其右孩，IN(X)\mathrm{IN}(X)IN(X) 为 XXX 结点的中序遍历，POST(X)\mathrm{POST}(X)POST(X) 为 XXX 结点的后序遍历。
则中序遍历 IN(X)=IN(L)XIN(R)\mathrm{IN}(X) = \mathrm{IN}(L)X\mathrm{IN}(R)IN(X)=IN(L)XIN(R)，
且后序遍历 POST(X)=POST(L)POST(R)X\mathrm{POST}(X) = \mathrm{POST}(L)\mathrm{POST}(R)XPOST(X)=POST(L)POST(R)X.
考虑到所有待处理的字符串都应为这一形式，自然而然地使用递归.

观察后序遍历 POST(X)\mathrm{POST}(X)POST(X)，我们发现最后一个字符为 XXX，因此对于待处理的字符串，可以在中序遍历中查找后序遍历的最后一个字符，将中序遍历 IN(X)\mathrm{IN}(X)IN(X) 自然而然地分为两部分，且显然有 IN(L)\mathrm{IN}(L)IN(L) 和 POST(L)\mathrm{POST}(L)POST(L) 长度相等，IN(R)\mathrm{IN}(R)IN(R) 和 POST(R)\mathrm{POST}(R)POST(R) 长度相等，可分离出左孩子和右孩子各自的中序遍历和后序遍历，递归调用即可.
