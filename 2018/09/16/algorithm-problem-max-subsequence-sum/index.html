Algorithm - Problem - 最大子段和

给出一段序列 aaa，选出其中连续且非空的一段使这段和最大，求这个最大的和.

分治

分析

分治的递归边界为单个元素，此时直接返回该元素的值.
对于数组 aaa 的 [l,r][l, r][l,r] 区间，其最大子段和的函数调用记为 SUB(l,r)\mathrm{SUB}(l, r)SUB(l,r).
递归调用时，m←(l+r)/2m\leftarrow (l+r)/2m←(l+r)/2，其最大子段和有三种情况

  1. 最大子段和出现在 [l,m][l, m][l,m] 区间.
  2. 最大子段和出现再 [m+1,r][m + 1, r][m+1,r] 区间.
  3. 最大子段和跨越了 mmm.

我们可以分别计算三种情况，并取最大值返回，以下是三种情况的计算方法

  1. 递归调用 SUB(l,m)\mathrm{SUB}(l, m)SUB(l,m).
  2. 递归调用 SUB(m+1,r)\mathrm{SUB}(m + 1, r)SUB(m+1,r).
  3. 求左区间以 a[m]a[m]a[m] 为尾的最大子段和 s1s_1s1​ 以及右区间以 a[m+1]a[m + 1]a[m+1] 为首的最大子段和 s2s_2s2​，则此时的值为 s1+s2s_1+s_2s1​+s2​.

于是 [l,r][l, r][l,r] 区间上的最大子段和为 max⁡{SUB(l,m),SUB(m+1,r),s1+s2}\max\{\mathrm{SUB}(l, m), \mathrm{SUB}(m + 1, r), s_1+s_2\}max{SUB(l,m),SUB(m+1,r),s1​+s2​}.

复杂度分析

T(n)=2T(2n)+O(n)T(n) = 2T(\frac{2}{n}) + \mathcal{O}(n)T(n)=2T(n2​)+O(n)，时间复杂度为 O(nlog⁡n)\mathcal{O}(n\log n)O(nlogn)

分析

最大子段和 XXX 实际上是求

X=max⁡{∑k=ija[k]∣∀i,j:0⩽i⩽j⩽n−1}.X=\max\left\{\sum_{k=i}^{j}a[k]\Bigg|\forall i,j:0\leqslant i\leqslant j\leqslant n-1\right\}. X=max{k=i∑j​a[k]∣∣∣∣∣​∀i,j:0⩽i⩽j⩽n−1}.

为方便起见，此处假定下标从 111 开始，以 nnn 结束，转为

X=max⁡{∑k=ija[k]∣∀i,j:1⩽i⩽j⩽n}.X=\max\left\{\sum_{k=i}^{j}a[k]\Bigg|\forall i,j:1\leqslant i\leqslant j\leqslant n\right\}. X=max{k=i∑j​a[k]∣∣∣∣∣​∀i,j:1⩽i⩽j⩽n}.

此处有两个变量 iii 和 jjj 变化，我们可以根据 max⁡\maxmax 的性质作出一些变化.
假设有 a,b,c,d,e,fa, b, c, d, e, fa,b,c,d,e,f 六个变量，显然 max⁡{a,b,c,d,e,f}=max⁡{max⁡{a,c},max⁡{b,d},max⁡{e,f}}\max\left\{a,b,c,d,e,f\right\} = \max\left\{\max\left\{a, c\right\}, \max\left\{b, d\right\}, \max\left\{e, f\right\}\right\}max{a,b,c,d,e,f}=max{max{a,c},max{b,d},max{e,f}}.
也就是说，我们可以对要求的值进行分组，分别求每个组内的最大值后，再求所有组的最大值.
我们不妨将所有上标 jjj 相同的值，分为一组，则转化为

X=max⁡{max⁡{∑k=ija[k]∣j=1,∀i:1⩽i⩽j},⋯,max⁡{∑k=ija[k]∣j=n,∀i:1⩽i⩽j}}.X=\max\left\{\max\left\{\sum_{k=i}^{j}a[k]\Bigg|j = 1, \forall i : 1\leqslant i\leqslant j\right\}, \cdots, \max\left\{\sum_{k=i}^{j}a[k]\Bigg|j = n, \forall i : 1\leqslant i\leqslant j\right\}\right\}. X=max{max{k=i∑j​a[k]∣∣∣∣∣​j=1,∀i:1⩽i⩽j},⋯,max{k=i∑j​a[k]∣∣∣∣∣​j=n,∀i:1⩽i⩽j}}.

我们不妨令

b[j]=max⁡{∑k=ija[k]∣∀i:1⩽i⩽j}b[j] = \max\left\{\sum_{k=i}^{j}a[k]\Bigg|\forall i : 1\leqslant i\leqslant j\right\} b[j]=max{k=i∑j​a[k]∣∣∣∣∣​∀i:1⩽i⩽j}

则有

X=max⁡{b[1],b[2],⋯,b[n]}=max⁡{b[j]∣∀j:1⩽j⩽n}X = \max\left\{b[1], b[2], \cdots, b[n]\right\} = \max\left\{b[j]\big|\forall j : 1\leqslant j \leqslant n\right\} X=max{b[1],b[2],⋯,b[n]}=max{b[j]∣∣​∀j:1⩽j⩽n}

观察 b[j]b[j]b[j]，其代表的实际上是以 a[j]a[j]a[j] 为尾的最大子段和，而求 b[j]b[j]b[j] 我们有两种决策

  1. b[j]=b[j−1]+a[j]b[j] = b[j - 1] + a[j]b[j]=b[j−1]+a[j].
  2. b[j]=a[j]b[j] = a[j]b[j]=a[j].

因此我们有递推式

b[j]=max⁡{b[j−1]+a[j],a[j]}.b[j] = \max\left\{b[j-1]+a[j], a[j]\right\}. b[j]=max{b[j−1]+a[j],a[j]}.
