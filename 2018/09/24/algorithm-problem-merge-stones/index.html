[NOI1995] 石子合并 洛谷 P1880

问题描述

在一个圆形操场的四周摆放 NNN 堆石子,现要将石子有次序地合并成一堆.规定每次只能选相邻的 222 堆合并成新的一堆，并将新的一堆的石子数，记为该次合并的得分。

试设计出算法,计算出将 NNN 堆石子合并成 111 堆的最小得分和最大得分.

输入

第1行正整数N,1⩽N⩽100N, 1\leqslant N\leqslant 100N,1⩽N⩽100,表示有 NNN 堆石子.
第2行有NNN个数,分别表示每堆石子的个数.

输出

输出共2行,第1行为最小得分,第2行为最大得分.

设石子数目存储于数组 aaa 中，下标从 000 开始.
对于此类环形问题，大多都可采用“化环为链”的手段，即 ∀i:0⩽i<n,a[i+n]=a[i]\forall i:0\leqslant i< n, a[i + n] = a[i]∀i:0⩽i<n,a[i+n]=a[i]，将原先 aaa 的所有元素拷贝一份追加到 aaa 最后，将环形转化为了线性结构.
此类问题是比较经典的区间 DP，因此我们设 dmin[i][j]d_{min}[i][j]dmin​[i][j] 表示从下标 iii 的石子到下标 jjj 的石子合并得分的最小值，类似地 dmax[i][j]d_{max}[i][j]dmax​[i][j] 表示从下标 iii 的石子到下标 jjj 的石子合并得分的最大值.
设 s[i][j]s[i][j]s[i][j] 为下标 iii 的石子到 下标 jjj 的石子的加和.
