0-1 背包

问题描述

有一个背包最大承载质量为 MMM，现在 nnn 种物品，第 iii 种物品的质量为 mim_imi​，价值为 viv_ivi​，每种物品只有 111 个，求能放的最大价值.

分析

假设 d[j]d[j]d[j] 表示最大承载质量为 jjj 时能放的最大价值，ddd 全部初始化为 000.

多重背包

问题描述

有一个背包最大承载质量为 MMM，现在 nnn 种物品，第 iii 种物品的质量为 mim_imi​，价值为 viv_ivi​，数量有 aia_iai​ 个，求能放的最大价值.

分析

个人一般是将多重背包转化为 0-1 背包，只需要将多个同种物品拆分为多种物品，即可转化为 0-1 背包.
