发表于 2018-10-20 | 分类于 Algorithm ， Problem

坐标跳跃

问题描述

在一个平面坐标系上，小明的初始位置坐标为 (p,q)(p,q)(p,q)，我们可以通过一次操作将他的位置变为 (p+q,q)(p+q,q)(p+q,q) 或 (p,p+q)(p, p+q)(p,p+q)。

给定一正整数 nnn，小明的初始位置为 (1,1)(1, 1)(1,1)，问最少需要多少次操作，可变为一个新的坐标 (x,y)(x, y)(x,y)，且其中的 xxx 或 yyy 至少有一个数字为 nnn 。
